## Installation

Install NPM package [@brandup/ui](https://www.npmjs.com/package/@brandup/ui-kit).

```
npm i @brandup/ui-kit@latest
```

## InputUIElement

`InputUIElement` - это базовый абстрактный класс для элементов пользовательсково ввода в форме.
